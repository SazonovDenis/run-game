### Скачиваем аудиофайлы

1.

Проверить, что установлен `python3` и `pip`

```
sudo apt install python3
sudo apt install python3-pip
```

2.

Проверить, что установлены библиотеки

```
pip install requests
pip install beautifulsoup4
```

3.

Выполнить скачиване

```
cd data/web-grab
./grab-mp3.sh
```
